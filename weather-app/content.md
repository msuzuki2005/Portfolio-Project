## Overview

Weather App was built to practice API integration and responsive UI design. The goal was a forecast experience that feels immediate and uncluttered.

## Challenge

Most weather interfaces pack too much data above the fold. Users wanted current temperature and the next few hours at a glance.

## Solution

The design centers on a mobile-first layout with location search, a prominent current conditions readout, and a scrollable hourly strip.

## Outcome

The app was deployed on Netlify. Building it reinforced how clarity comes from restraint — showing less data, but showing it better.

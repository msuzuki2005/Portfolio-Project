App Design

Meal Prep Planning App

A mobile/web app designed for planning weekly meals, prep schedules, and weekly grocery lists all in one place.

Client
ARTGR 4840/5840
Role
Designer, Developer
Tools
Cursor · Figma · Netlify · GitHub
View live site

This meal prep planning app was designed to reduce the friction of weekly meal planning for people who want to eat well without spending hours deciding what to cook.

Challenge

Users struggled to coordinate recipes, ingredients, and prep time across the week. Existing apps felt cluttered or required too much manual input before delivering value.

Process

Define

Mapped user flows/tasks, descriptive web structure, and site map.

Ideate

Collected and researched UI references, then sketched concepts for screen layouts, nav options, and mobile/web configurations.

Design System

Built from the Meal Prep Planner design system — color, type, spacing, controls, and cards.

Prototyping

Created a usable prototype using Cursor, Figma and Figma Make. App hosting was done with Netlify and GitHub.
